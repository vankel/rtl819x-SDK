Objective
=========
	This patch used to refine some changes for SDK v3.2.2.1.
	Before apply this patch, you must have already installed rtl819x-sdk-v3.2.tar.gz,rtl819x-sdk-v3.2.1.tar.gz,rtl819x-SDK-v3.2.1.1-20120605.tar.gz,rtl819x-SDK-v3.2.2.tar.gz and rtl819x-SDK-v3.2.2.1.tar.gz.
	
Apply SDK patch
===============
	- Please uncompress "rtl819x-sdk-v3.2.3-20121221.tar.gz" at rtl819x.
	- Follow 'rtl819x-sdk-v3.2/INSTALL.txt' to rebuild image.

Changes
=========
	Please refer to RTL819X_Firmware_Release_Note_v3.2.3.pdf for more details.

Note
=====
1.      How to check image version.
		At console, please input commands as follows:
		cat /proc/version
		cat /etc/version
